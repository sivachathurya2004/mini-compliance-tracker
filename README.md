# Mini Compliance Tracker

## Features
- View clients
- View tasks
- Add tasks
- Update status
- Highlight overdue tasks

## Tech Stack
- Node.js
- Express
- PostgreSQL
- HTML/CSS/JS

## Setup

1. cd backend
2. npm install
3. node server.js

4. Open frontend/index.html

## Assumptions
- No authentication
- Static clients

## Tradeoffs
- Simple UI
- No advanced filtering
